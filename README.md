MFNudity is an iOS library that can be helpful for checking image nudity. You can use it in any iOS application which is dealing with images.

*Swift 4.0 (Xcode 9.0)*

```ruby
pod 'MFNudity'
```

First Step  - `@import MFNudity` to your project
